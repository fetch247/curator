# 🚀 Coffee Bar Connoisseur - Simple Setup Guide

## Quick Start

### 1. Initial Git Setup (One Time Only)
```bash
git init
git remote add origin https://github.com/fetch247/curator.git
git config user.name "Your Name"
git config user.email "your.email@example.com"
```

### 2. Easy Updates (Every Time You Make Changes)

#### **Option A: Use the Sync Script**
- **Windows**: Double-click `sync-to-git.bat`

#### **Option B: Manual Commands**
```bash
git add .
git commit -m "Update website"
git push origin master
```

## 🎯 What This Does

1. **Adds all your changes** to Git
2. **Commits them** with a timestamp
3. **Pushes to GitHub** to make them live
4. **Updates your website** instantly

## 🔧 Troubleshooting

### If Git Commands Don't Work:
1. Make sure you're in the correct folder
2. Run the initial setup commands above
3. Check that your GitHub repository exists

### If Push Fails:
1. Make sure you have internet connection
2. Verify your GitHub credentials
3. Check that you have permission to push to the repository

## 📁 Files You Can Edit

- **`index.html`** - Your main website
- **`README.md`** - Documentation
- **Any other files** - Just run the sync script after changes
